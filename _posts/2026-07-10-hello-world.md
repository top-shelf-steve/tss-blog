Welcome to **rocvoc** — now living on GitHub Pages instead of micro.blog. Every post is just a markdown file: write it, `git push`, and the site updates itself.

## How a post works

Create a file in `_posts/` named `YYYY-MM-DD-your-title.md`, give it a few lines of front matter at the top (like this post has), and write markdown below. The first paragraph becomes the excerpt on the home page.

## Images work too

Drop an image into `assets/images/` and reference it like this:

![The rocvoc avatar]({{ "/assets/images/avatar.svg" | relative_url }})

That's the whole workflow. This post is safe to delete once the real ones arrive.
